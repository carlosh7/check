# Crear evento

> **Versión:** v12.44.0+ | **Módulo:** Eventos

## ¿Para qué sirve?
Crear un nuevo evento en el sistema. Cada evento es independiente: tiene su propia lista de invitados, configuración, ruleta, encuestas y más.

## Requisitos previos
- [ ] Tener una cuenta con rol **ADMIN** o **PRODUCTOR**
- [ ] (Opcional) Tener grupos/empresas creados para asignar el evento

## Pasos
1. Inicia sesión en Check Pro
2. En el panel principal, haz clic en **"Nuevo Evento"**
3. Completa los campos obligatorios:
   - **Nombre del evento** — ej: "Conferencia Anual 2026"
   - **Ubicación** — ciudad o lugar físico
   - **Fecha** — fecha y hora de inicio
4. (Opcional) Completa campos adicionales:
   - **Descripción** — texto informativo para los invitados
   - **Empresa/Grupo** — si trabajas con grupos, asigna el evento a uno
   - **Espacio (Venue)** — selecciona una ubicación registrada
5. Haz clic en **"Guardar"**

## Qué sigue
Una vez creado, el evento aparece en tu lista. Puedes:
- [Configurar sus ajustes](02-configurar-evento.md) (registro público, colores, emails)
- [Importar invitados](../02-invitados/01-importar.md)
- [Crear una ruleta](../03-ruleta/01-crear-sorteo.md)

## Notas
- Los eventos se crean con estado **ACTIVE** por defecto
- Cada evento puede tener su propia base de datos independiente (para eventos grandes)
- Puedes cambiar la fecha y ubicación después desde Ajustes
