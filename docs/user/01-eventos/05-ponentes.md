# Ponentes del evento

> **Versión:** v12.44.691+ | **Módulo:** Eventos

## ¿Para qué sirve?
Gestionar los ponentes, speakers o conferencistas de tu evento. Cada ponente puede tener nombre, tema, biografía, foto y redes sociales.

## Requisitos previos
- [ ] Evento creado

## Pasos
1. Ve a **Configuración del Evento** > pestaña **"Ponentes"**
2. Haz clic en **"Agregar ponente"**
3. Completa los campos:
   - **Nombre** — del ponente
   - **Tema / Charla** — título de su presentación
   - **Biografía** — breve descripción
   - **Foto (URL)** — enlace a una imagen
   - **Redes sociales** — Twitter, LinkedIn, sitio web
   - **Orden** — posición en la lista
4. Guarda
