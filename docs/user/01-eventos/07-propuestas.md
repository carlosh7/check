# Propuestas públicas

> **Versión:** v12.44.699+ | **Módulo:** Eventos

## ¿Para qué sirve?
Los invitados pueden proponer temas, charlas o actividades para el evento. El organizador revisa y aprueba las propuestas.

## Para el organizador
1. Ve a **Configuración del Evento** > pestaña **"Propuestas"**
2. Verás todas las propuestas enviadas por los invitados
3. Puedes cambiar el estado: **Pendiente → Aprobar / Rechazar**
4. Las propuestas aprobadas se muestran en la landing del evento

## Para los invitados
Los invitados pueden enviar propuestas desde la landing page del evento.
