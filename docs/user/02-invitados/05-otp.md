# Check-in por OTP

> **Versión:** v12.44.700+ | **Módulo:** Invitados

## ¿Para qué sirve?
Alternativa al check-in por código QR. Genera un código numérico de 6 dígitos para cada invitado. El invitado proporciona el código y el organizador lo verifica.

## Generar código OTP
1. Ve a la tabla de asistentes en el evento
2. Busca al invitado
3. Haz clic en el botón **OTP** junto al nombre
4. Se genera un código de 6 dígitos
5. Comparte el código con el invitado (por SMS, WhatsApp o email)

## Verificar OTP
1. En la barra de herramientas del evento, haz clic en **OTP**
2. Ingresa el código de 6 dígitos
3. El sistema muestra el nombre del invitado y marca su check-in

## Notas
- Cada código OTP es único por invitado
- El código se puede regenerar las veces que sea necesario
- El código no expira hasta que se usa o se genera uno nuevo
