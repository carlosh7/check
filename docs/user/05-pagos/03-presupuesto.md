# Presupuesto del evento

> **Versión:** v12.44.690+ | **Módulo:** Eventos

## ¿Para qué sirve?
Llevar el control de gastos de un evento. Registra conceptos, montos y categorías para tener visibility del presupuesto total.

## Requisitos previos
- [ ] Evento creado

## Pasos
1. Ve a **Configuración del Evento** > pestaña **"Presupuesto"**
2. El total acumulado se muestra en la parte superior
3. Haz clic en **"Agregar gasto"** para añadir una partida
4. Completa:
   - **Concepto** — descripción del gasto
   - **Monto** — valor numérico
   - **Categoría** — tipo de gasto (Catering, Espacio, Producción, etc.)
   - **Notas** — detalles adicionales
5. Guarda
