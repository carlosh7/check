# Branding del evento

> **Versión:** v12.44.689+ | **Módulo:** Eventos

## ¿Para qué sirve?
Personalizar la apariencia visual de un evento específico: colores, logo, CSS y HTML personalizado.

## Requisitos previos
- [ ] Evento creado
- [ ] Rol **ADMIN** o **PRODUCTOR**

## Pasos
1. Ve a **Configuración del Evento** > pestaña **"Branding"**
2. Configura las opciones:
   - **Color primario** — color principal del evento (botones, acentos)
   - **Logo URL** — URL pública del logo
   - **CSS Personalizado** — código CSS que se inyecta en las páginas públicas
   - **Header / Footer HTML** — HTML personalizado para el registro público
3. Haz clic en **"Guardar Branding"**

## Notas
- Los cambios solo afectan a este evento, no al sistema completo
- El CSS personalizado se inyecta en las páginas de registro público y landing
- El header y footer aparecen en el formulario de registro
