# Cambiar idioma

> **Versión:** v12.44.695+ | **Módulo:** Sistema

## ¿Para qué sirve?
Cambiar el idioma de la interfaz de Check Pro. Actualmente disponibles: Español e Inglés.

## Pasos
1. Ve a **Sistema** > pestaña **"Perfil"**
2. En la sección **"Idioma / Language"**, haz clic en el idioma deseado
3. La interfaz se actualiza inmediatamente
4. La preferencia se guarda para futuras sesiones

## Idiomas disponibles
| Idioma | Código | Botón |
|--------|--------|-------|
| Español | es | 🇪🇸 Español |
| English | en | 🇬🇧 English |

## Notas
- El cambio de idioma es instantáneo (no requiere recargar)
- La preferencia se guarda en el navegador
- Cada usuario puede tener un idioma diferente
